# **CRUD Laundry Crafty**
**Dyah Utami Kesuma Dewi - 5025241186**
https://dyahukd.blogspot.com/2025/12/learning-crud.html

  Pada pertemuan ke-11, saya mengerjakan sebuah proyek kecil berupa aplikasi layanan laundry berbasis web. Aplikasi ini dirancang untuk mencatat data pelanggan serta proses transaksi laundry dengan memanfaatkan database MySQL. Melalui tugas ini, kami diperkenalkan pada implementasi fungsi CRUD yang biasa digunakan dalam pengelolaan data pada sistem informasi.

  Untuk menjalankan aplikasi secara lokal, saya memanfaatkan XAMPP yang berfungsi sebagai server pengembangan. Pengaturan database dilakukan melalui phpMyAdmin.
