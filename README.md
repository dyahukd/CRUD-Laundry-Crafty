# **Laundry Crafty**
Laundry Crafty merupakan sebuah sistem informasi sederhana berbasis web.
Laundry Crafty juga merupakan contoh aplikasi web CRUD sederhana dengan PHP dan MySQL.
Laundry Crafty digunakan untuk melakukan pendataan terhadap laundry masuk dan laundry keluar. Laundry Crafty dibangun menggunakan PHP, Bootstrap, dan MySQL.
